# Assets Directory

This directory should contain the following assets for the Expo application:

## Required Assets

- `icon.png` - App icon (1024x1024px recommended)
- `splash.png` - Splash screen image (1242x2436px recommended)
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `favicon.png` - Web favicon (48x48px recommended)

## Creating Assets

You can use online tools or design software to create these assets:

1. **App Icon**: Create a 1024x1024px PNG with your app logo
2. **Splash Screen**: Create a 1242x2436px PNG with your branding
3. **Adaptive Icon**: Create a 1024x1024px PNG (Android will handle the shape)
4. **Favicon**: Create a 48x48px PNG for web

## Temporary Solution

For development, you can use placeholder images or create simple colored squares as temporary assets.

## Expo Asset Generation

You can also use Expo's asset generation tools:
- [Expo Asset Generator](https://docs.expo.dev/guides/app-icons/)
